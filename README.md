# msc

To run the LLM chatbot, you start navigating to the Recommender folder, and installing all necessary libraries in requirements.txt.

The LLM chatbot is stored in llm_chatbot.py. Because this application uses OpenAI's GPT-5nano, you will need to create a .txt file called openai_key.txt with your API key.

Once you run llm_chatbot.py, you will be prompted for a UserID. Biobank's data is priviliged, so a simulated entry is included as an example, with ID 1234567. After this ID is entered, please ask the prompted questions to obtain personalized recommendations.
